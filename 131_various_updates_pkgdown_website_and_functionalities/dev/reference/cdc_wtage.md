# CDC Weight-for-age-chart

Weight-for-age charts, 2 to 20 years

## Usage

``` r
cdc_wtage
```

## Format

A data frame with 436 rows and 5 variables:

- `SEX`:

  Sex: 1 = male, 2 = female

- `AGE`:

  Age in months

- `L`:

  Box-Cox transformation for normality

- `M`:

  Median

- `S`:

  Coefficient of variation

## Source

<https://www.cdc.gov/growthcharts/percentile_data_files.htm>
