# Find closest bin

Find closest bin

## Usage

``` r
find_closest_bin(param, breaks, labels)
```

## Arguments

- param:

  Parameter

  Vector containing parameter that needs closest bin checking

  Default value

  :   none

- breaks:

  Breaks created in
  [`get_bins()`](https://pharmaverse.github.io/admiralpeds/dev/reference/get_bins.md)

  Default value

  :   none

- labels:

  Labels created in
  [`get_bins()`](https://pharmaverse.github.io/admiralpeds/dev/reference/get_bins.md)

  Default value

  :   none

## Value

detects appropriate bin
