# WHO Length/height-for-age for boys

WHO Length/height-for-age charts for boys from day 0 (birth) to day 1856

## Usage

``` r
who_lgth_ht_for_age_boys
```

## Format

A data frame with 1857 rows and 4 variables:

- `Day`:

  Age in days

- `L`:

  Box-Cox transformation for normality

- `M`:

  Median

- `S`:

  Coefficient of variation

## Source

<https://www.who.int/tools/child-growth-standards/standards/length-height-for-age>
